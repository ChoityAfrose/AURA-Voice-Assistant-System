# AURA-Voice-Assistant-System

**AURA** is an intelligent Python-based AI voice assistant built to simplify your daily digital interactions. Powered by speech recognition and natural text-to-speech responses, AURA provides a fully hands-free experience‚Äîallowing you to control your computer, retrieve information, and perform tasks using just your voice.

Designed as a personal productivity companion, **AURA** can open applications, browse the web, search Google or Wikipedia, play music, tell jokes, and engage in small talk. It listens actively, understands user commands, and responds in a natural, conversational manner‚Äîmaking your workflow smoother, faster, and more enjoyable.

**AURA** aims to combine the functionality of modern AI assistants with the interactive feel of fictional systems like JARVIS, creating a responsive and customizable assistant that you can expand with new features over time. Whether you're automating repetitive tasks or simply exploring AI interaction, AURA is your personal companion for an intelligent, voice-driven experience.

## üõ† Features

- Recognize voice commands using Google Speech Recognition

- Speak responses using pyttsx3

- Time & Date announcements

- Wikipedia search with spoken summary

- Open websites like Google, Facebook, YouTube

- Open Google and perform searches through voice commands.

- Close websites like Google

- Play random music from a specified folder and stop the currently playing song.

- Open system applications: Calculator, Notepad, CMD

- Close system applications: Calculator, Notepad, CMD

- Open Calendar (Google Calendar via browser)

- Tell jokes and respond to basic small talk

- Take screenshot and save it to a folder.

- Take today‚Äôs notes, show the notes, and delete the notes as part of my daily routine.

- Works as a study assistant by explaining topics and giving quizzes.

- Control windows - Minimize, Maximize and Restore by voice command

- Exit gracefully with a voice command

## How to run? 

1. Create a virtual environment:

   ```bash
   conda create -n aura python=3.11 -y
      ```
2. Activate virtual environment:

   ```bash
   conda activate aura
   ```
3. Install required packages:

   ```bash
   pip install -r requirements.txt
   ```
4. Run the JARVIS script:

   ```bash
   python aura.py
   ```
5. For exe file conversion

    ```bash
   pip install pyinstaller
   ```
